This repo is a complete starter for OneTool. It is ready to import into GitHub/Codespaces and deploy to Vercel.

## Features
- Next.js 14 (App Router)
- Tailwind CSS (PostCSS)
- Header, Drawer, Footer components
- AI demo page (client demo)
- Tools, Learn, About, Privacy, Terms pages
- Firebase placeholder lib for future integration
- public/index.html fallback (for simple static hosters)
- Vercel-ready (auto deployment)
- env.example showing required environment variables

## Run locally (Codespaces or laptop)
1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000

## Deploy
- Push to GitHub and import into Vercel (1-click). Set env vars on Vercel dashboard.
